Partial correlation (control for nodes): gives you the possibility to create association
matrices based on partial correlations using the subject’s time series signals. You have the
option to compute partial correlations considering all nodes in the network but also to
select specific nodes to control for when computing the correlation matrix. When selecting
this checkbox you will be reminded of this by a help dialog “Please select nodes from
network node menu in the network construction panel”. When constructing a network based
on partial correlations you will have to be very careful with a post-hoc redefinition of your
network (which will still be possible but may be methodologically invalid). To start
generating the matrices just click and refer to the “Generate Correlation Matrix” option.
